# Moodify 🎵✨
<img src="img/MoodifyLogo.png" alt="Logo Moodify" width="200"/>

**Moodify** es una aplicación de análisis de emociones a partir de imágenes, utilizando **Machine Learning** y **Deep Learning**. La app permite predecir el estado de ánimo de una persona mediante una foto y recomienda música acorde a cómo te sientes.

---

## 🌟 Motivación del proyecto

El objetivo de **Moodify** es explorar la intersección entre la inteligencia artificial y la música, ofreciendo una experiencia personalizada y emocional al usuario. El proyecto surge de la necesidad de:  

- Comprender emociones a partir de imágenes de manera automatizada.  
- Integrar predicciones emocionales con recomendaciones musicales.  
- Crear una aplicación accesible y rápida utilizando modelos optimizados.  

---

## 🚀 Características principales

- Clasificación automática de canciones según emociones detectadas.  
- Modelos entrenados con **TeachableMachine**. 
- Interfaz interactiva con **Streamlit**.  
- Compatible con archivos de audio y datasets de canciones.  
- Visualización de gráficos y métricas de predicción en tiempo real.  
- Integración con la **API de Spotify** para recomendaciones personalizadas.  

---

## 🛠 Tecnologías utilizadas

- **Python 3.10**  
- **TeachableMachine**  
- **Streamlit**  
- **Git** y **GitHub** para control de versiones
